=== KNOWN COMPATIBILITY ISSUES & RESOLUTIONS ===

Issue: Prisma Client generation fails
Resolution:
  npm run prisma:generate
  Clear node_modules: rm -rf node_modules && npm ci

Issue: Sharp installation fails on Windows
Resolution:
  Already configured - prebuilt binaries included
  No build tools required

Issue: TypeScript strict mode errors
Resolution:
  Already configured in tsconfig.json
  Use "as any" only for external libraries if needed

Issue: CORS blocking payment gateway
Resolution:
  Ensure backend CORS_ORIGIN matches frontend domain
  Whitelist all required payment domains

Issue: JWT token expired immediately
Resolution:
  Check JWT_EXPIRE format: "7d", "24h", etc.
  Ensure server time is synchronized

=== DOCKER COMPATIBILITY (Optional) ===

If deploying with Docker:
  Base image: node:18-alpine (lightweight)
  Alpine OS: 40MB vs 150MB for node:18
  Production image size: ~250-300MB

Dockerfile reference:
  FROM node:18-alpine
  WORKDIR /app
  COPY package*.json ./
  RUN npm ci --only=production
  COPY . .
  RUN npm run build
  EXPOSE 3000
  CMD ["npm", "start"]

  5. Restart application
  6. Verify health check endpoints
