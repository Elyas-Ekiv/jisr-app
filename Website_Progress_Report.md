## 1. Project Overview

This report outlines the current progress of the Jisr Web Platform (MVP) development. The focus of this phase was completing the front-end design, user interfaces, and core functionality for all user types (Parents, Children, and Administrators).

Jisr is an Augmentative and Alternative Communication (AAC) platform designed to help children with autism communicate through visual images, symbols, and text-to-speech technology. The platform bridges the communication gap between children and their surroundings.

- **Lucide React** - Icon library
- **Vite 5.0.0** - Build tool and development server
